# דף נחיתה - איריס קורן

## אודות הפרויקט

דף נחיתה מקצועי עבור איריס קורן, מטפלת EFT מוסמכת המציעה סדנאות לצוותי חינוך והוראה.

הדף בנוי על בסיס התוכן מה-PDF שסופק ומכיל את כל הפרטים על השירותים המוצעים.

## שליחת אימיילים מהטופס

✅ **הטופס מוכן ועובד עם SendGrid!**

### Current Setup:
- **Domain**: iriscoren.co.il
- **Email**: iris@iriscoren.co.il
- **Frontend**: index.html + script-sendgrid.js
- **Backend**: Node.js server עם SendGrid API

### Quick Start:
1. **Setup SendGrid** - ראה `QUICK_SETUP_iriscoren.md`
2. **Deploy to Hetzner** - העתק קבצים והרץ server
3. **Configure Nginx** - הפנה API calls לport 3000
4. **Test** - שלח טופס ובדוק אימייל

### קבצים עיקריים:
- `index.html` - הדף הראשי (מוגדר לSendGrid)
- `script-sendgrid.js` - JavaScript עם SendGrid integration  
- `server/` - Backend Node.js עם SendGrid
- `QUICK_SETUP_iriscoren.md` - מדריך מהיר! ⭐

## הרצת הדף מקומית

1. פתח טרמינל בתיקיית הפרויקט
2. הרץ שרת HTTP פשוט:
   ```bash
   python3 -m http.server 8000
   ```
3. פתח דפדפן וגש לכתובת: `http://localhost:8000`
